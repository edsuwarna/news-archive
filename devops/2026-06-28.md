## 1. 🖥️ **AWS Lambda Introduces MicroVMs — Fully Isolated Serverless Sandboxes**
AWS launched a new serverless compute primitive, Lambda MicroVMs, providing VM-level isolated sandboxes with no shared kernel or resources between sessions. Each sandbox has rapid launch/resume, full lifecycle control, and a writable filesystem — bridging the gap between Lambda's existing execution model and EC2-level isolation for security-sensitive workloads.
**Source:** https://aws.amazon.com/blogs/aws/run-isolated-sandboxes-with-full-lifecycle-control-aws-lambda-introduces-microvms/
