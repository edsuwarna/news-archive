## 1. 🖥️ AWS Lambda Introduces MicroVMs — Isolated Sandboxes with Full Lifecycle Control

AWS launched a new serverless compute primitive: Lambda MicroVMs. These are VM-level isolated sandboxes with no shared kernel or resources between sessions, providing rapid launch and resume, full lifecycle control, state preservation up to 8 hours, and no infrastructure to manage. Built on Firecracker, this bridges the gap between serverless and full VM isolation.

**🔗** https://aws.amazon.com/blogs/aws/run-isolated-sandboxes-with-full-lifecycle-control-aws-lambda-introduces-microvms/

## 7. 🔄 Kubernetes v1.36: Mixed Version Proxy Graduates to Beta

The Mixed Version Proxy (MVP) — first introduced as alpha in K8s 1.28 — has graduated to beta. This feature makes cluster upgrades dramatically safer by intelligently proxying API requests between control plane versions during rolling updates, preventing "unknown version" incompatibility errors during multi-version control plane deployments.

**🔗** https://kubernetes.io/blog/2026/06/kubernetes-v1-36-mixed-version-proxy-beta/

## 📋 Summary: Week's Major Themes

- **🔮 Agentic Infrastructure Goes Mainstream** — This week's dominant theme: almost every major announcement (AWS Blocks, Azure Agentic Cloud Ops, Cloudflare Agent Skills, Equinix's "ClickOps is Dead") revolves around AI agents operating on infrastructure, not just generating code. Platform engineering is pivoting from "self-service portals" to "agent-service APIs."

- **☸️ Kubernetes Matures as the Universal Control Plane** — K8s v1.36's Mixed Version Proxy and PSI GA, combined with Microsoft AKS adding bare metal and fleet management, reinforce Kubernetes as the unifying layer across VM, bare metal, and AI accelerator infrastructure.

- **⚡ Linux Kernel Performance Renaissance** — Linux 7.2 delivers transformative performance improvements: 360% MySQL gain from Cache Aware Scheduling, 30-100% MongoDB gains from MGLRU, and universal +12% from GCC tuning. For SREs and platform engineers, upgrading the kernel offers immediate infrastructure-level returns.

- **🔐 Post-Quantum & AI-Supply Chain Security** — The launch of Akrites and Linux 7.2's post-quantum ML-DSA signatures signal a dual focus: protecting OSS infrastructure from AI-accelerated attacks while also future-proofing against quantum threats.

- **🏗️ Custom Silicon Reshapes Cloud Economics** — OpenAI's Jalapeño ASIC and Arm's Top500 milestone highlight a structural shift: general-purpose CPUs and GPUs are being supplemented (and potentially displaced) by specialized chips designed for specific AI and cloud workloads, changing the cost calculus for infrastructure teams.
