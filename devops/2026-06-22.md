### 8. ⚙️ Kubernetes v1.36 Ships: Volume Snapshots GA, PSI Metrics GA, Mixed Version Proxy Beta

### 12. ⚠️ AgentJacking: A Single Sentry Error Can Hijack Claude Code, Cursor & Codex
Security researchers demonstrated a critical vulnerability dubbed "AgentJacking" where a fake Sentry error notification can hijack AI coding agents (Claude Code, Cursor, Codex) through the MCP (Model Context Protocol) channel. By injecting a malicious payload into what appears to be a standard error report, attackers can execute arbitrary code on the developer's machine. This is a wake-up call for MCP security in the AI-assisted development ecosystem.
🔗 https://thenewstack.io/agentjacking-sentry-mcp-attack/
