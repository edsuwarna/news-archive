## 1. 🔬 AWS Replaces Fat-Tree Data Center Networks with Random Graph Theory, Cutting Routers by 69%
AWS disclosed that Resilient Network Graphs (RNG), a flat network architecture based on quasi-random graph theory, is now the default for most new data center builds. Replacing traditional fat-tree topologies with random graph wiring reduces router count by 69% while maintaining or improving performance — a massive infrastructure engineering achievement that fundamentally changes data center networking economics.

**Source:** https://www.infoq.com/news/2026/06/aws-random-graph-data-center/

## 20. 🎛️ Kubernetes v1.36: Mixed Version Proxy Graduates to Beta
The Mixed Version Proxy (MVP) for Kubernetes API Server has graduated to beta in Kubernetes v1.36. MVP enables safer cluster upgrades by proxying API requests between control planes of different versions during rolling upgrades — reducing the risk of API incompatibility during the most dangerous operation in cluster management. Multiple other v1.36 features also landed this week (PSI Metrics → GA, Volume Group Snapshots → GA, Workload-Aware Scheduling, Service ExternalIPs deprecation).
