1. **💰 GitHub Copilot Switches to Usage-Based Pricing — Users Hit with Sticker Shock**
GitHub moved Copilot from request-based billing to an AI-credit model, with $10/month Pro plans getting 1,500 credits ($15 worth). Power users report burning through their entire monthly allotment in a single day — one complex prompt cost 700 credits, and a couple of Copilot-led commits ate 5,000 credits. The pricing shift has sparked widespread debate about the true cost of AI-assisted coding at scale.

2. **📂 AWS Launches S3 Files — Buckets Accessible as High-Performance File Systems**
AWS announced S3 Files, making S3 buckets mountable as POSIX-compatible file systems with ~1ms latencies on EC2 and other compute. This eliminates the historic tradeoff between object storage durability and file system interactivity — a massive quality-of-life improvement for data engineers, ML teams, and anyone running stateful workloads on cloud storage.
🔗 https://aws.amazon.com/blogs/aws/launching-s3-files-making-s3-buckets-accessible-as-file-systems/

5. **☸️ Kubernetes v1.36 Ships: PSI Metrics GA, Volume Snapshots GA, Mixed Version Proxy Beta**

## Summary Themes

- **🔮 The Agentic Cloud is Real**: Multiple vendors (Cloudflare, Google GKE, AWS Bedrock) shipped infrastructure specifically designed for AI agents — sandboxed execution environments, autonomous account creation, and SRE AI copilots. This is the week's dominant theme.

- **💰 AI Pricing Realities Hit Home**: GitHub Copilot's usage-based pricing rollout and the resulting user backlash signals a broader reckoning: as AI inference costs become visible to end users, the era of unlimited subsidized AI coding is ending. Platform engineers need to budget for AI credits the way they budget for compute.

- **☸️ Kubernetes Platform Maturation**: K8s v1.36's GA features (PSI metrics, Volume Snapshots), GKE's standby buffer and active buffer, and the Agent Sandbox launch all point to Kubernetes evolving from container orchestration into a full-fledged platform for all workloads — including AI agents.

- **🔐 Supply Chain Security Stays Critical**: The Red Hat NPM backdoor attack is yet another reminder that software supply chain security isn't optional. SBOM generation, signature verification, and dependency auditing should be part of every CI/CD pipeline.

- **🔧 Infrastructure Observability is Leveling Up**: Between Cloudflare's Town Lake + Skipper platform, Google's AI-in-SRE work, and the debate over logs/metrics/traces vs. behavioral observability, the O11y space is evolving fast — moving from dashboards to AI-driven diagnosis and business-outcome measurement.
