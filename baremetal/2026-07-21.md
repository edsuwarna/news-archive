## 6. 🔗 Linux "KNOD" Patches Enable In-Kernel Network Offloading to AMD GPUs

Groundbreaking Linux kernel patches for "KNOD" (Kernel Network Offload Device) allow direct in-kernel network offloading to AMD GPUs — no ROCm userspace needed. Posted as an RFC by Taehee Yoo, this enables GPU-accelerated packet processing, virtual switching, and network functions directly from the kernel. For bare-metal server operators, this could dramatically reduce CPU overhead for network-intensive GPU workloads.

**Source:** https://www.phoronix.com/news/KNOD-Network-Offload-AMD-GPUs

## 7. 🦾 Arm Core Local Accelerator Driver Lands for Linux

Arm announced the Core Local Accelerator (CLA) Linux driver — an agnostic interface for programming attached accelerators directly from the CPU. Targeted initially at attached compute engines, this positions ARM server platforms for tighter accelerator integration without vendor lock-in. It's a significant step for the ARM server ecosystem as it scales beyond general-purpose workloads into AI and HPC acceleration.

**Source:** https://www.phoronix.com/news/Arm-Core-Local-Accelerator

## 8. 🚀 MLPerf Training v6.0: Lambda Delivers Fastest LLM Training on GB300 NVL72

## 📊 Bare Metal Trend Summary

- **Memory becomes a battleground** — With DDR5-8000 RDIMMs and MRDIMM Gen2 hitting 12,800 MT/s, server memory speeds are accelerating faster than CPU generations. Mid-life platform memory upgrades (Intel Xeon 6) are becoming a competitive necessity.
- **Direct liquid cooling goes mainstream** — MSI's EPYC Venice DLC node proves DLC is no longer confined to supercomputers. Expect every new server platform to offer DLC as a standard option, not a special-order niche.
- **GPU BMaaS competition is measurable** — MLPerf v6.0's 8-team GB300 NVL72 results transform benchmark performance into a market signal. Bare-metal GPU providers now have public, comparable training numbers — expect this to drive pricing and SLAs.
- **ARM server infrastructure matures** — The Arm Core Local Accelerator driver and Holo Core ARM64 Linux builds show the ARM server ecosystem is building the software foundation for widespread adoption beyond storage and web serving.
- **Networking innovation moves to the kernel** — KNOD's in-kernel GPU network offloading hints at a future where bare-metal servers handle accelerated networking entirely within the kernel, bypassing userspace frameworks and reducing latency for AI cluster interconnects.
