This is a landmark moment for ARM in bare-metal HPC — a pure CPU system outperforming GPU-accelerated competitors at the very top of the list — and signals ARM's maturation as a first-class server architecture for dense, high-performance compute.

## 3. 🖥️ Lambda Launches Bare Metal Instances — DPU-Powered, No Hypervisor

Lambda released **Bare Metal Instances**, a new BMaaS offering that combines full, unvirtualized hardware access with API-driven lifecycle management. Powered by **NVIDIA BlueField DPUs**, these instances spin up in minutes via Lambda's cloud API with zero hypervisor overhead — no IPMI, no PXE, no manual provisioning. This is a significant leap forward in bare-metal cloud: it brings cloud-class automation to truly dedicated hardware, closing the gap between traditional dedicated servers and public cloud VMs for GPU-intensive workloads.

📎 https://lambda.ai/blog/lambda-bare-metal-instances

## 4. ⚡ Lambda Dominates MLPerf Training v6.0 — Fastest GB300 NVL72 & MoE Benchmarks

Lambda posted the **fastest LLM training time on NVIDIA GB300 NVL72** in MLPerf Training v6.0, training Llama 3.1 8B 18.7% faster than the nearest competitor, and the **fastest Mixture-of-Experts (MoE) training on HGX B200**. ## 5. 🔌 NVIDIA's Co-Packaged Optics Switch Unboxed — 3 kW Saved Per Switch

Lambda unboxed one of the first **NVIDIA Quantum-X InfiniBand Photonics CPO (co-packaged optics) switches**, revealing concrete power savings: **3.05 kW saved per switch** vs traditional pluggable transceivers, scaling to ~4.4 MW in a 41,472-GPU cluster. CPO eliminates the most failure-prone component in GPU cluster networking — the pluggable optical module — while slashing power draw and latency. This is a transformative development for bare-metal GPU fabric TCO and reliability, moving liquid-cooled CPO from experimental to production-ready.

📎 https://lambda.ai/blog/unbox-one-of-nvidias-first-co-packaged-optics-samples-with-lambda

Equinix's latest infrastructure analysis tackles the fundamental architecture question facing every bare-metal operator: should you **scale up** (bigger, denser nodes) or **scale out** (more nodes, distributed)? For the bare-metal world, this is the ultimate edge computing deployment: radiation-hardened, thermally extreme, latency-constrained, and fully autonomous. The storage architecture is a fascinating mix of commercial-off-the-shelf (COTS) and custom hardening.

## 📊 Bare Metal Trend Summary

- **🔹 GPU BMaaS enters the DPU era** — Lambda's BlueField DPU-powered Bare Metal Instances show that true cloud-class automation on unvirtualized hardware is now production-ready. Expect copycat offerings from CoreWeave, Nebius, and others within 6-12 months.
- **🔹 Co-packaged optics (CPO) is real and shipping** — 3+ kW saved per switch, fewer failure points. CPO is the most impactful networking trend for GPU cluster TCO since NVLink. Every hyperscale GPU deployment should be evaluating CPO fabric.
- **🔹 ARM in the data center has crossed a threshold** — ARM-powered #1 Top500 system (LineShine, 2.2 EFLOPs pure CPU) + NVIDIA Vera benchmarks maturing. ARM is no longer a niche server architecture; it's competing for top-tier HPC and BMaaS workloads.
- **🔹 AI silicon is fragmenting fast** — Qualcomm (Dragonfly + HBM alternative), Broadcom (Jalapeño with OpenAI), and NVIDIA all racing for the data center AI compute wallet. Bare-metal procurement teams face a more complex chip landscape than ever.
- **🔹 Regulation is driving repatriation to physical infrastructure** — EU data sovereignty pressures and DORA compliance are pushing financial services and critical infrastructure back toward colocation and dedicated servers within sovereign borders. Bare-metal demand from compliance-driven enterprise is rising.
