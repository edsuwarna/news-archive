## 1. 🔥 Lambda Launches Bare Metal Instances with NVIDIA GB300 NVL72 — No Hypervisor, Full API Control

Lambda unveiled its **Bare Metal Instances** product, debuting on the NVIDIA GB300 NVL72 Supercluster. Unlike traditional dedicated servers that rely on IPMI/PXE-based provisioning, Lambda's approach uses NVIDIA BlueField DPUs in zero-trust mode to deliver a fully API-driven instance lifecycle — launch, terminate, attach storage — with no third-party hypervisor between the OS and the metal. Customers get direct, unmediated access to CPU, memory, GPU, disk, and TPM. This marks a significant maturation of the GPU BMaaS space, bridging the gap between cloud convenience and bare-metal performance.

**Source:** https://lambda.ai/blog/lambda-bare-metal-instances

## 2. ⚡ NVIDIA Co-Packaged Optics Switch Unboxed by Lambda — 3.05 kW Saved Per Switch

Lambda published a deep-dive unboxing of the **NVIDIA Quantum-X InfiniBand Photonics Q3450-LD**, one of the first co-packaged optics (CPO) switches in production sampling. The 4U switch delivers 115.2 Tb/s non-blocking switching while consuming only 3.95 kW — versus 7.0 kW for a traditional switch with pluggable transceivers. At a 41,472-GPU cluster scale, CPO frees up 4.4 MW of power — equivalent to running 3,137 extra GPUs. The switch uses UDQ4 liquid cooling and 48V DC busbar power, same infrastructure model as NVIDIA GB300 NVL72 racks. CPO removes ~655,000 discrete transceiver failure points from a 128K-GPU fabric.

**Source:** https://lambda.ai/blog/unbox-one-of-nvidias-first-co-packaged-optics-samples-with-lambda

## 4. 🖥️ Gigabyte Shows 40-Node 1U Cluster at Computex — 320 Cores, 1.28 TB RAM, 80 SSDs

**Gigabyte's R1C7-K0A-AS1** is a 40-node cluster crammed into a single 1U chassis, on display at Computex 2026. Each sled holds 8 nodes, each with 8 CPU cores, 32 GB of memory, 2× SSDs, and an integrated GPU — totaling 320 cores, 1.28 TB of RAM, and 80 drives in one rack unit. This is a density monster targeting edge deployments, HPC clusters, and high-density virtualization where physical space is the constraint. The modular sled architecture also simplifies field maintenance — pull one sled to service 8 nodes without disturbing the rest.

**Source:** https://www.servethehome.com/a-40-node-1u-cluster-gigabyte-r1c7-k0a-as1/

## 5. 📈 RISC-V CPU Performance Up 8× in Five Years — Phoronix Benchmarks

Phoronix published a comprehensive retrospective showing **8× performance improvement** in RISC-V CPUs over five years, comparing the 2019 SiFive HiFive Unmatched against the 2025 SpacemiT K3 (first-to-market RVA23 SoC). The K3 Pico-ITX mini computer now competes respectably against entry-level x86 in single-threaded workloads, while multi-threaded gains have been even more dramatic thanks to wider cores and deeper pipelines. With Fedora 44 now offering official RISC-V images and Linux 7.2 adding ESWIN SoC support by default, the RISC-V server ecosystem is building real momentum for bare-metal deployments.

**Source:** https://www.phoronix.com/review/risc-v-5-year-performance

As geopolitical factors push more Chinese enterprises toward domestic hardware, Hygon's growing ecosystem maturity is worth monitoring for anyone tracking server CPU diversity.

## 10. 🔧 Linux 7.2 To Add ACPI CPPC v4 Support Authored by NVIDIA — Ahead of Vera Rubin

NVIDIA engineers contributed **ACPI CPPC v4 (Collaborative Processor Performance Control)** support to the upcoming Linux 7.2 kernel, ahead of the Vera Rubin hardware ramp. CPPC is the standard mechanism for OS-driven CPU frequency scaling on server hardware, and v4 brings improved power efficiency controls for high-core-count server CPUs. This is an early enablement signal for NVIDIA's next-generation server platform — bare-metal operators running Vera Rubin-based infrastructure will need this support for optimal power and thermal management.

**Source:** https://www.phoronix.com/news/ACPI-CPPC-v4-Linux-7.2

## 📊 Bare Metal Trend Summary

- **🔥 GPU BMaaS enters the DPU era** — Lambda's Bare Metal Instances using BlueField DPUs represent a new category: cloud-class API-driven dedicated servers without hypervisor overhead. Expect copycats from CoreWeave, Vultr, and others as the pattern proves out.

- **🔌 Co-packaged optics go operational** — NVIDIA CPO switches are no longer a lab curiosity. Lambda's unboxing shows real-world power savings of 3 kW/switch, with production deployment implications for every large GPU cluster builder. This will reshape bare-metal networking TCO calculations.

- **🦾 RISC-V server momentum keeps building** — 8× performance in 5 years, Fedora 44 RISC-V images shipping, ESWIN SoC support default in Linux 7.2. The RISC-V bare-metal ecosystem is transitioning from "experimental" to "deployable for specific workloads."

- **📦 Density madness continues** — Gigabyte's 40-node 1U cluster shows that physical density innovation hasn't peaked. For edge and HPC bare-metal deployments, squeezing more compute per rack unit remains a key differentiation.

- **🏗️ AI infrastructure funding floods the market** — Lambda's $1B upsized facility is just one signal. The sheer scale of capital flowing into GPU bare-metal capacity means supply is catching up to demand, which could drive BMaaS pricing down over the next 12-18 months.
