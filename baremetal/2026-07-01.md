## 1. 🖥️ Lambda Launches Bare Metal Instances — Full Hardware Control, API-Driven, No Hypervisor
Lambda has launched **Bare Metal Instances** combining direct, unmediated hardware access with cloud-grade API-driven lifecycle management. Powered by **NVIDIA BlueField DPUs**, the instances deliver no hypervisor overhead, TPM-backed security, and full control over CPU, memory, GPU, and disk. Starting with the **NVIDIA GB300 NVL72** and Lambda Supercluster, this is a direct BMaaS play for AI/ML teams that need both raw hardware performance and DevOps-style provisioning.

**Source:** https://lambda.ai/blog/lambda-bare-metal-instances

## 2. 🚀 AMD EPYC 8005 "Sorano" — The Biggest Server CPU Shake-Up Since Rome
AMD's **EPYC 8005 "Sorano"** uses the same SP6 socket as the EPYC 8004 "Siena" but completely transforms the performance profile — up to **84 cores / 168 threads**, 384MB L3 cache, **DDR5-6400** memory support, and **CXL 2.0**. It's a drop-in upgrade for existing SP6 platforms (with BIOS update), delivering ~10% higher system-level power consumption for significantly more cores and performance. ServeTheHome calls it "the biggest upheaval of a server product line since the EPYC 7002 Rome launch."

**Source:** https://www.servethehome.com/amd-epyc-8005-sorano-completely-changes-amd-sp6/

## 3. 🔗 NVIDIA Co-Packaged Optics Switches Unboxed — 44% Less Network Power, Fewer Failure Points
Lambda unboxed one of the first **NVIDIA Quantum-X InfiniBand Photonics Q3450-LD CPO switches** — a breakthrough in data center networking. CPO moves optical conversion next to the switch ASIC, eliminating pluggable transceivers (which number ~655,000 in a 128K-GPU cluster). The Q3450-LD draws **3.95 kW vs 7.0 kW** for a standard switch. At 41,472 GPUs, this frees **4.4 MW** of power — equivalent to **+3,137 extra GPUs** in the same facility envelope. Liquid-cooled with 48V DC busbar power.

**Source:** https://lambda.ai/blog/unbox-one-of-nvidias-first-co-packaged-optics-samples-with-lambda

## 8. 🔬 IBM Unveils Sub-1nm Nanostack Transistor — Chips Built Vertically
IBM outlined its **Nanostack transistor architecture** for the **sub-1nm era** (targeting the 2030s). Rather than shrinking transistors laterally, Nanostack builds upward — stacking transistor layers vertically via wafer bonding. This extends Moore's Law beyond traditional lithography limits and has direct implications for future server CPUs, GPUs, and AI accelerators. If successful, this could enable server processors with dramatically higher transistor density than current 3nm/2nm nodes.

**Source:** https://www.servethehome.com/ibm-outlines-sub-1nm-nanostack-transistor-technology/

## 10. 🏗️ GCC 17 Lands Support for SpacemiT X100 — RISC-V Server Core Progress
The **GCC 17 compiler** has added targeting support for the **SpacemiT X100** RISC-V core, marking continued progress in the RISC-V server ecosystem. While still early compared to x86 and ARM, this compiler enablement is a necessary step before RISC-V server hardware can become viable for bare-metal deployments. SpacemiT's X100 targets high-performance computing workloads and could eventually offer an alternative to proprietary architectures in data centers.

**Source:** https://www.phoronix.com/news/GCC-17-SpacemiT-X100

## 🔥 Bare Metal Trend Summary

- **GPU Bare-Metal Goes Mainstream** — Lambda's new Bare Metal Instances and MLPerf dominance on GB300 NVL72 signal that the BMaaS model for AI/ML workloads has arrived. API-driven provisioning + direct hardware access is becoming the standard ask from AI teams.

- **Power & Cooling Are the New Battlegrounds** — From liquid-cooled 800V DC busbars to NVIDIA CPO switches cutting network power by 44%, every layer of the data center stack is being re-engineered for the thermal and electrical demands of 1000W+ accelerators.

- **Data Sovereignty = Bare-Metal Opportunity** — European regulations and geopolitical tensions are driving cloud repatriation. Colocation and bare-metal deployments offer jurisdictional control that hyperscale cloud can't always guarantee.

- **Server CPU Landscape in Flux** — AMD's EPYC 8005 "Sorano" delivers 84 cores in a drop-in SP6 socket, while RISC-V server targets (SpacemiT X100) get compiler enablement. The 2030s outlook includes IBM's Nanostack sub-1nm vertical transistors — the next architectural revolution in chip fabrication.

- **Networking Interconnect Evolution** — CPO (co-packaged optics) switches from NVIDIA remove 655K+ failure points from large GPU clusters and reclaim megawatts of power for actual compute. This is the infrastructure innovation that makes 100K-GPU clusters economically viable.
