## 🏗️ **ASF (Apache Software Foundation)**

### Top-Level Project Promotions
- **Apache Polaris Graduates** 🌟 — Open source data catalog for Iceberg, graduated from incubator to Top-Level Project. Polaris 1.4 & 1.5 released with advanced security, metrics, and federation.
- **Apache Gluten Graduates** 🌟 — Native vectorized execution engine for Spark, also promoted to Top-Level Project.

### Major Releases
- **Apache Iceberg 1.11.0** — REST Catalog GA and encryption features for the open table format.
- **Apache HTTP Server 2.4.68** — Security release addressing use-after-free, DoS, XSS, and buffer overflow flaws.
- **Apache Polaris 1.5** — Open source data catalog enhancements.
- **Apache Kafka** — Dimster benchmarking tool and streaming changes tooling emerging in the ecosystem.
- **Apache Arrow** — nanoarrow (lightweight Arrow implementation) gaining traction.

### Security Advisories
- **CVE-2026-23918** ⚠️ — Critical HTTP/2 double-free flaw in Apache HTTP Server, enabling DoS and potential RCE.
- **Apache ActiveMQ Vulnerability** — Security header injection attack vector disclosed.
- **Apache Tomcat Flaws** — EncryptInterceptor bypass vulnerabilities reported.

### Foundation News
- **ASF Launches Responsible AI Initiative** 🤖💚 — $10M initiative with initial $1.75M donation from **Anthropic**. Focus on responsible AI development practices.
- **Anthropic Donates $1.5M to ASF** — Supporting the broader Apache open source ecosystem.
- **Community Over Code (ApacheCon)** — Upcoming events in the ASF conference calendar.
- **New HTTP/2 "Bomb" Vulnerability** — Impacts Apache, NGINX, IIS, Envoy, and others — remote DoS via HPACK bomb.

## 📡 **OpenInfra Foundation** (formerly OpenStack Foundation)

### 🚨 Major Consolidation
- **OpenInfra Foundation joins the Linux Foundation** 🏛️ — In a historic move, the OpenInfra Foundation (home of OpenStack, Kata Containers, StarlingX, Zuul, Airship) is now part of the Linux Foundation ecosystem. This comes 15 years after OpenStack's launch. The move signals a new chapter for open infrastructure.

### Releases
- **OpenStack 2026.1 "Gazpacho" Released** 🥘 — Focus areas:
  - **VMware replacement** — Migration tooling for organizations moving off VMware
  - **Simplified operations** — Reduced complexity for operators
  - **Performance improvements** — Faster networking and compute
- **OpenStack Flamingo** (previous release) — Paid down significant technical debt.

### Community
- **OpenStack Deployments Growing** — The community marks **15 years** of OpenStack. Deployments continue to climb.
- **New Strength in Resilience** — OpenStack finds renewed relevance as organizations seek open alternatives to proprietary cloud.

### Projects Under OpenInfra Umbrella
- **Kata Containers** — Continued development as a secure container runtime.
- **StarlingX** — Edge computing platform updates.
- **Zuul** — CI/CD platform.
- **Airship** — Lifecycle management for infrastructure.
