## 🌐 **CNCF (Cloud Native Computing Foundation)**

### 🚀 Latest Releases & Updates

- **Kubernetes v1.36.2** released (12 Jun) — the latest stable patch for the 1.36 release train. Alpha work on **v1.37** is already underway (v1.37.0-alpha.2, 25 Jun).
- **Helm 4 Goes GA 🎉** — Helm v4.2.2 (17 Jun) marks a **major milestone** for the Kubernetes package manager. Helm 4 brings significant architectural improvements over Helm 3, including new plugin system and enhanced security defaults. This is the biggest Helm release in years.
- **Prometheus v3.13.0** (1 Jul) — latest major release of the CNCF-graduated monitoring system with continued performance improvements.
- **Flux v2.9.0** (30 Jun) — the GitOps tool hits a new version with enhanced multi-cluster support and improved OCI artifact management.
- **etcd v3.6.13** (1 Jul) — security and stability update for the distributed key-value store.
- **gRPC v1.82.0** (2 Jul) — new release of the high-performance RPC framework with various improvements.
- **Istio 1.30.2** (24 Jun) + **Istio 1.28.10** (1 Jul) — dual patch releases for the service mesh. 1.30 continues Istio's ambient mesh evolution.
- **OpenTelemetry Collector v0.155.0** (23 Jun) — continuing rapid iteration on the observability pipeline.
- **Backstage v1.52.1** (26 Jun) — the CNCF-incubating developer portal keeps shipping.
- **K3s v1.36.2+k3s1** (24 Jun) — lightweight Kubernetes distribution updated alongside upstream K8s.
- **Crossplane v2.3.3** (22 Jun) — cloud-native control plane framework continues to mature.
- **cert-manager v1.20.3** (25 Jun) — cloud-native certificate management update.
- **Dapr 1.18** (11 Jun) — introduced **Verifiable Execution**, a groundbreaking feature enabling cryptographic attestation of Dapr application execution, marking a big step for trust in distributed application runtime.
