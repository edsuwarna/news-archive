## 🟡 High/Medium

### CVE-2026-3865 — CSI Driver SMB Path Traversal (CVSS 7.1) 🔥 NEW
- **Date:** April 11, 2026
- Insufficient validation of the `subDir` parameter in volume identifiers allows path traversal. A malicious user with PV creation privileges can craft volumeHandle containing traversal sequences to delete/modify directories outside the intended managed subdirectory within the SMB export.
- **Fixed in:** CSI Driver for SMB >= v1.20.1
- **Reporter:** Shaul Ben Hai (SentinelOne)
- **Detection:** Look for traversal sequences like `../` in volumeHandle fields

### CVE-2026-4342 — ingress-nginx Comment Injection (CVSS 8.8) 🔥 NEW
- **Date:** March 20, 2026
- A combination of Ingress annotations can be used to inject configuration into nginx via comments. Leads to arbitrary code execution in the ingress-nginx controller and disclosure of Secrets.
- **Fixed in:** ingress-nginx v1.13.9 / v1.14.5 / v1.15.1
- **Reporter:** wooseokdotkim

### CVE-2026-3864 — CSI Driver NFS Path Traversal (CVSS 7.1) 🔥 NEW
- **Date:** March 17, 2026
- Insufficient validation of the `subDir` parameter in volume identifiers allows path traversal (same pattern as CVE-2026-3865).
- **Fixed in:** CSI Driver for NFS >= v4.13.1
- **Reporter:** Shaul Ben Hai (SentinelOne)

### CVE-2026-3288 — ingress-nginx rewrite-target Injection (CVSS 8.8) 🔥 NEW

## 🔑 Key Trends

1. **ingress-nginx is #1 threat vector** — 8 CVEs in 2025-2026 alone, including 5 CRITICAL (CVSS 9.8) and multiple HIGH (8.8). Every K8s admin should prioritize ingress-nginx upgrades.
2. **CSI Driver path traversal** — Two new CVEs (SMB + NFS) reported by SentinelOne follow the same `subDir` traversal pattern. These require PV creation privileges but allow host filesystem modification.
3. **Supply chain attacks targeting K8s infrastructure** — TeamPCP's active campaigns (Trivy, LiteLLM, Checkmarx, GitHub) specifically include Kubernetes lateral movement toolkits in their payloads.
4. **Unfixed architectural CVEs** — The K8s SRC has officially designated 4 CVEs as unfixed (CVE-2020-8561/8562, CVE-2021-25740, CVE-2020-8554). Scanners will flag these starting June 1.
5. **Cloud-native credential theft** — PCPJack and ongoing campaigns target exposed Docker, K8s, Redis, and MongoDB services worm-style.
6. **Fluent Bit and telemetry infrastructure** — Five new Fluent Bit vulnerabilities (RCE, auth bypass, path traversal) show telemetry pipelines as expanding attack surface.
