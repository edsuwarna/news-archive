## 1. 🔴 Critical & High Severity CVEs

### ingress-nginx Comment-Based Configuration Injection
**CVE-2026-4342** · 🔴 **HIGH (8.8)** · Published: 2026-03-20
- Annotations can be used to inject nginx configuration → arbitrary code execution in the ingress-nginx controller
- Disclosure of Secrets accessible to the controller (cluster-wide in default installation)
- **Affected:** ingress-nginx < v1.13.9, < v1.14.5, < v1.15.1
- **Fixed:** v1.13.9, v1.14.5, v1.15.1
- Reported by wooseokdotkim

### ingress-nginx Rewrite-Target Configuration Injection
**CVE-2026-3288** · 🔴 **HIGH (8.8)** · Published: 2026-03-10
- The `nginx.ingress.kubernetes.io/rewrite-target` annotation can inject nginx configuration
- Arbitrary code execution + Secret disclosure
- **Affected:** ingress-nginx < 1.13.8, < 1.14.4, < 1.15.0
- **Fixed:** 1.13.8, 1.14.4, 1.15.0
- Can be partially mitigated by blocking rewrite-target annotations via admission control

### Multiple ingress-nginx Vulnerabilities (Batch Disclosure)
**CVE-2026-1580, CVE-2026-24512, CVE-2026-24513, CVE-2026-24514** · 🔴 **HIGH (8.8)** · Published: 2026-02-02
- Four CVEs disclosed simultaneously in ingress-nginx
- **Affected:** ingress-nginx < v1.13.7, < v1.14.3
- **Fixed:** v1.13.7, v1.14.3
- Upgrade is required for full mitigation

## 2. 🟡 Medium Severity CVEs

### CSI Driver for SMB Path Traversal
**CVE-2026-3865** · 🟡 **MEDIUM (6.5)** · Published: 2026-04-11
- Insufficient validation of `subDir` parameter in volume identifiers allows path traversal
- Malicious user with PersistentVolume create privilege can delete/modify directories outside intended scope on SMB server
- **Affected:** CSI Driver for SMB < v1.20.1
- **Fixed:** v1.20.1+
- CVSS: `AV:N/AC:L/PR:H/UI:N/S:U/C:N/I:H/A:H`

### CSI Driver for NFS Path Traversal
**CVE-2026-3864** · 🟡 **MEDIUM (6.5)** · Published: 2026-03-17
- Same traversal pattern as CVE-2026-3865 but for NFS CSI driver
- **Affected:** CSI Driver for NFS < v4.13.1
- **Fixed:** v4.13.1+
- CVSS: `AV:N/AC:L/PR:H/UI:N/S:U/C:N/I:H/A:H`

## 5. 📊 Trends & Recommendations

### Dominant Attack Vectors
1. **Ingress-nginx remains the #1 attack surface** — 5 CVEs in 2026 alone, all rated HIGH (8.8). Upgrade aggressively and plan migration to Gateway API.
2. **CSI driver vulnerabilities** — Two path-traversal CVEs in CSI drivers (SMB + NFS) in 2026. Restrict PersistentVolume creation privileges.
3. **Supply chain attacks** — TeamPCP campaign demonstrates CI/CD pipeline compromise leading to warez-laced container images. Validate image provenance.
4. **Cloud credential theft** — PCPJack and Storm-1977 show credential harvesting at scale.
