## 🟡 Medium Severity

### CVE-2026-3865 — CSI Driver for SMB Path Traversal (Apr 10, 2026)
- **CVSS 6.5 | Medium**
- Insufficient validation of `subDir` parameter allows path traversal to delete/modify directories on SMB server
- Affected: smb.csi.k8s.io < v1.20.1
- Mitigation: upgrade to v1.20.1, restrict PersistentVolume creation
- *Source: K8s Security Announcement*

## ℹ️ Notable Items & Updates

### Reconciling the Past: Unfixed K8s CVEs Corrected (May 26, 2026)
The Kubernetes SRC announced corrections to CVE records for **four unfixed vulnerabilities** that were previously listed with incorrect "fixed version" metadata. The records were updated on **June 1, 2026**.

**Affected CVEs (all remain unfixed):**
| CVE | Issue | Mitigation |
|-----|-------|-----------|
| **CVE-2020-8554** | kube-apiserver follows HTTP redirects to admission webhooks — can expose internal networks | Set log level < 10; disable `--profiling` |
| **CVE-2020-8561** | TOCTOU race in API server proxy DNS resolution — IP restriction bypass | Use local DNS cache (dnsmasq) with `min-cache-ttl` |
| **CVE-2020-8562** | *Also addressed in the same correction* | See GitHub issues |
| **CVE-2021-25740** | Endpoints API allows cross-namespace forwarding via manually specified IPs | Restrict write access to Endpoints/EndpointSlices (not in default edit/admin roles since v1.22) |

**Impact:** Vulnerability scanners may now flag these CVEs in places they didn't before. Administrators should verify mitigation configurations are in place.

*Source: [Kubernetes Blog — Reconciling the Past](https://kubernetes.io/blog/2026/05/26/reconciling-unfixed-kubernetes-cves/)*

## 📈 Trends & Recommendations

1. **CVE Record Integrity** — The "Reconciling the Past" effort signals Kubernetes SRC is tightening CVE data quality. Expect fewer "false fixed" records going forward and more accurate vulnerability scanner results.

2. **TeamPCP Supply Chain Campaign** — The threat actor remains active, shifting from K8s-native tools (Trivy, KICS) to the broader developer ecosystem (GitHub, LiteLLM, Checkmarx). The GitHub breach is the largest single code exfiltration event of 2026.

4. **CSI Driver Security** — Two path traversal CVEs (SMB + NFS) highlight the importance of restricting PersistentVolume creation to trusted administrators.

5. **Docker AuthZ Bypass** — Incomplete fix for CVE-2024-41110 emphasizes that even previously-patched components need re-audit when new bypass techniques emerge.
