## 2️⃣ Security News

### 🔴 PCPJack Credential Stealer Exploits 5 CVEs to Spread Worm-Like Across Cloud Systems
**Source:** The Hacker News — May 7, 2026
A newly discovered credential theft framework called **PCPJack** targets exposed cloud infrastructure by chaining 5 CVEs. It harvests credentials from cloud, container, developer, productivity, and financial services, exfiltrating data through attacker-controlled infrastructure. The toolset actively removes artifacts of rival group TeamPCP from compromised environments, suggesting turf wars among cloud credential theft groups. **Relevant to K8s operators:** exposed kubeconfigs, cloud provider keys, and CI/CD secrets are primary targets.

### 🟡 Docker Engine CVE-2026-34040 — AuthZ Bypass (CVSS 8.8)
**Source:** The Hacker News — Apr 7, 2026
A high-severity vulnerability in Docker Engine allows attackers to bypass authorization plugins (AuthZ). Tracked as **CVE-2026-34040** (CVSS 8.8), this is an **incomplete fix for CVE-2024-41110**, a maximum-severity bug from July 2024. Impact: containers running with AuthZ plugins may have insufficient authorization enforcement. Docker Engine users should update immediately.

### 🔴 Trivy Hack Spreads Infostealer via Docker Hub — Worm + K8s Wiper
**Source:** The Hacker News — Feb 9, 2026
The **Trivy supply chain attack** (malicious versions 0.69.4–0.69.6 on Docker Hub) continues to have widening blast radius. The malware distributes an infostealer via Docker containers and includes **worm-like propagation** capabilities plus a **Kubernetes wiper** component. Last known clean release: Trivy 0.69.3. **Verify your Trivy images immediately.**

### 🔴 Chaos Mesh Critical GraphQL Flaws — Full K8s Cluster Takeover
**Source:** The Hacker News — Jul 18, 2025
Multiple critical GraphQL vulnerabilities in Chaos Mesh allow attackers with minimal in-cluster network access to achieve RCE and full Kubernetes cluster takeover. Attackers can execute fault injections (pod shutdowns, network disruptions) and escalate privileges.

### 🔴 NVIDIA Container Toolkit CVE-2025-23266 — Container Escape (CVSS 9.0)
**Source:** The Hacker News — May 6, 2025
Disclosed by Wiz, the "NVIDIAScape" vulnerability (CVE-2025-23266, CVSS 9.0) is a critical container escape in the NVIDIA Container Toolkit affecting managed AI cloud services. All platforms with NVIDIA Container Toolkit are affected.

### 🟡 Microsoft Warns: Default Helm Charts → Data Leaks
**Source:** The Hacker News — Apr 4, 2025
Microsoft Defender for Cloud researchers warn that out-of-the-box Helm charts prioritize ease of use over security, introducing misconfigurations that leak data. **Recommendation:** audit Helm chart defaults before deployment, especially for persistent volume claims, network policies, and Secrets.

### 🟡 Self-Spreading Dero Miner Infects Docker Containers

## 3️⃣ Trends & Recommendations

### 📈 Trend: Supply Chain Attacks on Container Tooling Accelerating
The **Trivy supply chain attack** (Feb 2026), following earlier incidents, shows attackers are weaponizing trusted container security tools themselves. The attack's inclusion of a **Kubernetes wiper** component signals a shift from pure cryptomining toward destructive capabilities. 

**Action:** Pin container image digests, verify signatures via cosign, monitor Docker Hub pull counts for anomalies.

### 📈 Trend: ingress-nginx Remains the Most Targeted K8s Component
Four new advisories in 2026 (CVE-2026-4342, CVE-2026-3288, plus the Feb batch of 4 CVEs) — ingress-nginx annotation injection continues to be the richest attack surface in the K8s ecosystem. Each allows RCE + secret disclosure.

**Action:** Upgrade to **ingress-nginx v1.12.1** (latest). Monitor annotation-based attacks via admission webhooks. Consider using a WAF in front of ingress-nginx.

### 📈 Trend: Container Escape + Cloud Credential Theft Converging
PCPJack + NVIDIA Container Toolkit CVE-2025-23266 + Docker AuthZ bypass CVE-2026-34040 — attackers are chaining container escapes with cloud credential theft to move laterally from containers to cloud provider APIs.

**Action:** Enforce workload identity (IRSA/Pod Identity), rotate cloud credentials frequently, and deploy runtime security monitoring (Falco, Tracee).

### 📈 Trend: Worm-Like Propagation Across Docker and K8s
PCPJack (May 2026), Trivy malware (Feb 2026), and the Dero miner (Apr 2025) all feature worm-like self-propagation. Exposed Docker API sockets and misconfigured kubeconfigs are the primary entry vectors.

**Action:** Block public access to Docker API (2375/2376) and K8s API server. Enforce network policies. Use admission controllers to prevent privileged containers.
